# School Backend (sin JWT, password plano)

## Pasos

1. npm install
2. Copia .env.example a .env y pon tu Postgres
3. npm run dev

## Endpoints

- GET /
- POST /api/auth/login
- GET /api/auth/me?userId=1
- GET /api/catalogos/niveles
- GET /api/catalogos/grados
- GET /api/catalogos/secciones/:gradeId
- POST /api/estudiantes
